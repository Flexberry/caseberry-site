Курсы

Для специалистов доступны следующие курсы:
№ Название курса
1 Caseberry Programming - объектно-ориентированная разработка программного обеспечения с использованием CASE-платформы Caseberry
2 Caseberry Analysis - объектно-ориентированный анализ с использованием CASE-платформы Caseberry и унифицированного языка моделирования UML
3 Caseberry Design - объектно-ориентированное проектирование программного обеспечения с использованием CASE-платформы Caseberry и унифицированного языка моделирования UML
4 Создание интеграционных компонент для корпоративной сервисной шины Highway SB
По окончании каждого курса выдаётся удостоверение о повышении квалификации (72 часа) установленного образца.

График проведения

Курсы проводятся в специализированном учебном классе по факту набора группы в количестве не менее 10 человек. Для уточнения стоимости и записи на курсы обращайтесь в раздел Контакты.
